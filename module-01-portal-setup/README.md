# Module 01 - Portal Setup & API Publishing

> **Scenario.** Your APIs are live behind Kong Gateway, but external partners and developers have no way to discover or subscribe to them. You'll create a Developer Portal on Konnect and publish your first API products.

## Module outcomes
- Create a Developer Portal on Konnect
- Publish API products with OpenAPI specs, versions, and implementations
- Link gateway services to API products
- Browse and test the live portal
