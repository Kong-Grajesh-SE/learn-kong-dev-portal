# Module 02 - App Registration & Auth

> **Scenario.** Your APIs are published on the portal, but developers can't authenticate. You'll configure key-auth for developer self-service registration and walk through the complete developer experience.

## Module outcomes
- Configure key-auth authentication strategy
- Walk through developer self-service registration
- Make authenticated API requests with app credentials
- Understand the developer experience end-to-end
