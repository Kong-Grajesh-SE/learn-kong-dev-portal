# Module 03 - Portal Customization & Teams

> **Scenario.** Your portal is functional but generic. You'll brand it with custom themes, add pages and snippets, configure team-based visibility, and set up SSO identity providers.

## Module outcomes
- Customize portal branding and theme
- Add pages, snippets, and API documents
- Configure team-based access control and visibility
- Set up SSO identity providers (OIDC/SAML)
